# fy
Minimal command line [ZHCN->ENG / ENG->ZHCN] translation tool, using [fanyi.baidu.com](http://fanyi.baidu.com)  

极简的命令行[中英/英中]翻译工具, 使用[fanyi.baidu.com](http://fanyi.baidu.com)接口  

开发框架: 
+ commons-io-2.5.jar
+ groovy-2.4.7.jar
+ groovy-json-2.4.7.jar
+ jodd-core-3.7.1.jar
+ jodd-http-3.7.1.jar
